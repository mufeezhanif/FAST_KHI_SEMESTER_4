Bash Scripting Basics

Special Environment Variables

VariableDescription
$0The name of the script itself
$1, $2, ...Positional parameters (arguments passed to the script)
$#The number of arguments passed to the script
$@All arguments passed to the script as separate words
$*All arguments passed to the script as a single string
$$The process ID (PID) of the current script
$?The exit status of the last executed command
$_The last argument of the previous command
$!The PID of the last background command

Loop with Condition

Illustrates a while loop that prints a counter until it exceeds 5.

counter=1
while [[ $counter -le 5 ]]; do

Basic If Statement

Simple Condition

Checks if 5 is greater than 3 using the extended [[ ]] syntax.

#!/bin/bash
    if [[ 5 -gt 3 ]]; then
        echo "5 is greater than 3"
        fi
5 is greater than 3
If-Else Statement

Conditional Branching

Demonstrates an if-else condition using [[ ]] for flexible syntax.

#!/bin/bash
        if [[ 2 -gt 3 ]]; then
            echo "2 is greater than 3"
            else
                echo "2 is not greater than 3"
                fi
2 is not greater than 3
If-Elif-Else Statement

Multi-Branch Decision

Handles multiple conditions using if, elif, and else with [[ ]] syntax.

#!/bin/bash
                if [[ 2 -gt 3 ]]; then
                    echo "2 is greater than 3"
                    elif [[ 2 -eq 2 ]]; then
                    echo "2 equals 2"
                    else
                        echo "Neither condition met"
                        fi
2 equals 2
Logical Operators in If Conditions

Combining Conditions

Uses && and || within [[ ]] to build complex conditional expressions.

#!/bin/bash
                        a=7
                        if [[ $a -gt 5 && $a -lt 10 ]]; then
                            echo "a is between 5 and 10"
                            else
                                echo "a is not between 5 and 10"
                                fi

                                b=3
                                if [[ $b -lt 2 || $b -gt 5 ]]; then
                                    echo "b is either less than 2 or greater than 5"
                                    else
                                        echo "b is between 2 and 5"
                                        fi
a is between 5 and 10
                                        b is between 2 and 5

Using a Case Statement

Case Handling

Uses a case statement to handle different inputs from the user.

#!/bin/bash
                                                    read -p "Enter a number between 1 and 3: " num
                                                    case $num in
                                                    1)
                                                    echo "You entered one."
                                                    ;;
                                                    2)
                                                    echo "You entered two."
                                                    ;;
                                                    3)
                                                    echo "You entered three."
                                                    ;;
                                                    *)
                                                    echo "Invalid input."
                                                    ;;
                                                    esac
Enter a number between 1 and 3: 2
                                                    You entered two.
